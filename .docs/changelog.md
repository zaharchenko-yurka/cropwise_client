# Журнал змін

## [Версія - в розробці] - 2026-03-01

### Додано

- Створено моделі даних (`dataclasses`) для основних ресурсів API: `Crop`, `Field`, `AgroOperation` в `models.py`.
- Реалізовано методи `from_api` та `to_api` для перетворення даних між форматом API та моделями.
- Додано обробку `datetime` об'єктів та опціональних полів в моделях.
- Додано нові приклади використання клієнта в `main.py`.

### Змінено

- Оновлено ендпоінти `CropsEndpoint`, `FieldsEndpoint`, та `OperationsEndpoint` для використання нових моделей даних замість необроблених JSON.
- Класи ендпоінтів тепер успадковують `WritableEndpoint` для підтримки операцій `create`, `update`, та `delete`.
- Методи `create` та `update` в ендпоінтах тепер приймають об'єкти моделей.
- Покращено типізацію в моделях та ендпоінтах.
- Оновлено файл `main.py` для коректного виклику методів клієнта.
- Стандартизовано імена методів в ендпоінтах (`list` замість `get_all`, `get` замість `get_by_id`) для відповідності базовим класам.
